# Durosport – India's Sports Infrastructure Partner

Official website for Durosport — design, consultation, and turnkey construction of world-class sports facilities across India.

## Structure
- `index.html` — Single-page website (all sections)
- `images/` — Project images (to be added)

## Sections
- Hero with animated brand visual
- Stats bar (200+ projects, 15+ sports, 50+ cities)
- Services (6 cards)
- Sports we cover (16 sports with SVG icons)
- Our Process (4 steps)
- Featured Projects (5 real projects)
- Surface Types
- Why Durosport
- CTA
- Contact
- Footer

## Brand
- **Primary:** Deep Blue #003380, Sport Green #00875A
- **Accents:** Turf Green #36B37E, Accent Teal #00A3A3
- **Typography:** Archivo (headings), Plus Jakarta Sans (body)

## Hosting
Deployed via GitHub Pages at `durosport.com`

## Adding Project Images
Replace gradient backgrounds in the Featured Projects section with actual image URLs:
```html
<div class="project-bg" style="background: url('images/your-image.jpg') center/cover no-repeat;"></div>
```
