# Durosport Website

Single‑page marketing website for Durosport.

## Files

- `index.html` — main site
- `robots.txt` — crawler rules
- `sitemap.xml` — sitemap for search engines
- `og-image.jpg` — social sharing image (1200×630)
- `durosport-favicon.svg` — favicon
- `durosport-app-icon.svg` — Apple touch icon
- `durosport-logo-primary.svg` — logo used in structured data

## Deploy

Host the repo root as a static site. Ensure HTTPS is enabled.

## Update OG Image

Replace `og-image.jpg` with a new 1200×630 image and keep the same filename.
